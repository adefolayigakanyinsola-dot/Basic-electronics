# Touch Enabled Circuit Design
A non microcontroller electronics project that lights up the LED when two insulated wires touch each other at the same time.
# Project Description
It is a simple project built on a breadboard. When two bare wires which serves as connectors are touched at the same time, a small amount of elevtricity passes through the skin due to low conductivity and high resistance. This in turn causes the transisitor to come on. The transistor then produces a large current from the base and battery combined which triggers the flow of current from the collector to the emitter.Then the current is passed into the LED to turn it on.
# INSTALLATION  PROCESS
# Software - KICAD
After creating a new file and nameing the circuit design,draw the schematics.
To do this,click on the component symbol icon on the top right icon bar to the side and select the components.
Use the mouse to drag and connect the wires accordingly.
Assign footprints by using the upper ico to the right on the menu bar or left click the mouse on each component,then navigate to footprints to check for suitable type of footprint component.
# PCB LAYOUT
Open the PCB bar by clicking on the default icon which is top right to the extreme,it takes you to thr layout that has all the components layout in PCB.
Using edge cuts in the apperance icon to the right,draw the small board outline enough to cover the circuit.
Arrange the components and usd the route signals to connect them 
Using the view icon in the top left icon on the menu bar,check the 3D layout.
